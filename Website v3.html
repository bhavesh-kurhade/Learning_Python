Multiplication Table

i = 1
while i <= 10:
    print(f"{num} * {i} = {num * i}")
    i += 1

for i in range(1, 11):
    print(f"{num} * {i} = {num * i}")

Control Statements: Break and Continue

Break (Stops at 5)

for i in range(1, 11):
    if i == 5:
        break
    print(i)

Continue (Skip some values)

# Skip 5
for i in range(1, 11):
    if i == 5:
        continue
    print(i)

# Skip 5 and 7
for i in range(1, 11):
    if i == 5 or i == 7:
        continue
    print(i)

File Handling

# Read file
